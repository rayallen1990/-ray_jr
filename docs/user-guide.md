# 用户指南

## 支持的命令

Ray_jr 知识库通过 `/kb` 前缀命令交互：

| 命令 | 说明 |
|------|------|
| `/kb ask <问题>` | 向知识库提问 |
| `/kb upload` | 上传文档到知识库（附带文件） |
| `/kb list` | 列出已上传的文档 |
| `/kb status` | 查看知识库状态 |
| `/kb sync` | 从 Git 仓库同步知识库 |

## 命令详解

### /kb ask — 知识库问答

向你的私有知识库提问，系统会检索相关文档片段并生成回答。

**用法：**
```
/kb ask 什么是PLC的工作原理？
```

**回复示例：**
```
PLC（可编程逻辑控制器）的工作原理是...

参考来源：
- 工业自动化基础.pdf (第3章)
- PLC编程手册.docx (第1节)
```

**说明：**
- 系统从你的私有命名空间检索最相关的 5 个文档片段
- 使用 Claude API 基于检索结果生成回答
- 每个用户的知识库相互隔离

### /kb upload — 上传文档

上传文件到你的私有知识库。支持 PDF、Word（.docx/.doc）和纯文本（.txt）格式。

**用法：**
1. 在聊天中发送 `/kb upload`
2. 同时附带要上传的文件

**支持的格式：**
- PDF（.pdf）— 最大 100MB
- Word（.docx, .doc）— 最大 100MB
- 纯文本（.txt）— 最大 100MB

**处理流程：**
1. 文件验证（格式、大小）
2. 文档解析（提取文本内容）
3. 文本分块（800字符/块，100字符重叠）
4. 向量嵌入生成（批量处理，每批32个）
5. 索引到 Qdrant 向量数据库

**回复示例：**
```
📄 正在解析文档 工业手册.pdf...
✓ 文档解析完成，共 45 个片段
⏳ 正在生成向量嵌入 (32/45)...
⏳ 正在生成向量嵌入 (45/45)...
✅ 文档已成功索引到知识库！

文档信息：
- 文件名：工业手册.pdf
- 大小：2.35 MB
- 片段数：45
- 文档 ID：a1b2c3d4e5f6
```

### /kb list — 列出文档

查看当前知识库中已上传的所有文档。

**用法：**
```
/kb list
```

**回复示例：**
```
📚 你的知识库文档：

1. 工业手册.pdf — 45 片段 — 2024-01-15
2. PLC编程指南.docx — 32 片段 — 2024-01-16
3. 安全规范.txt — 12 片段 — 2024-01-17

共 3 个文档，89 个片段
```

### /kb status — 知识库状态

查看知识库的运行状态和统计信息。

**用法：**
```
/kb status
```

**回复示例：**
```
📊 知识库状态：

- 状态：正常运行
- 文档数：3
- 总片段数：89
- 向量维度：1536
- 存储命名空间：tenant:dingtalk:user123:private
```

### /kb sync — 仓库同步

从配置的 Git 仓库同步知识库文档。管理员可以将文档放入 Git 仓库，系统自动拉取并索引。

**用法：**
```
/kb sync
```

**说明：**
- 从配置的 Git 仓库拉取最新文档
- 自动检测新增/修改/删除的文件
- 增量更新向量索引

## 多渠道接入

### 微信

通过微信公众号或企业微信接入，直接在聊天窗口使用 `/kb` 命令。

### 飞书

通过飞书机器人接入，支持：
- 单聊：私有知识库
- 群聊：共享知识库（群成员共享文档）

### 钉钉

通过钉钉机器人接入，支持：
- 单聊：私有知识库
- 群聊：共享知识库

## 多租户隔离

每个用户的知识库数据完全隔离：
- 命名空间格式：`tenant:{channel}:{user_id}:private`
- 群聊命名空间：`tenant:{channel}:group:{group_id}:private`
- 不同渠道的同一用户视为不同租户（防止跨渠道数据泄露）
- 匿名用户自动分配临时命名空间
